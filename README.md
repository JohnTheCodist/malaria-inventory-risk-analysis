Antimalarial stock-outs during peak malaria season can delay treatment, increase patient risk, and force pharmacies into costly emergency procurement.
This project analyzes historical pharmacy malaria data to determine whether malaria demand is seasonal and how inventory strategies can be optimized to reduce stock-out risk while avoiding year-round overstocking.

The analysis was inspired by a real-world debate on whether malaria cases are truly seasonal, and uses real pharmacy dispensing data to explore the answer empirically.
